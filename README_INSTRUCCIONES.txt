=====================================
     INSTRUCCIONES TODO LIST APP
=====================================

¡Hola! Esta aplicación te permitirá gestionar tus tareas diarias de forma sencilla.

PRIMERA VEZ - INSTALACIÓN:
==========================
1. Haz doble clic en: install_dependencias.bat
2. Espera a que termine la instalación (unos minutos)
3. Cuando termine, ya puedes usar la aplicación

USAR LA APLICACIÓN:
==================
1. Haz doble clic en: start.bat  
2. Espera a que aparezca: "Application startup complete"
3. Abre tu navegador web (Chrome, Firefox, Edge, etc.)
4. Ve a: http://localhost:8000
5. ¡Ya puedes crear y gestionar tus tareas!

CERRAR LA APLICACIÓN:
====================
1. En la ventana negra que se abrió, presiona: Ctrl + C
2. O simplemente cierra la ventana negra

DESINSTALAR LA APLICACIÓN:
=========================
Si ya no quieres usar la aplicación y quieres limpiar tu computadora:

1. Haz doble clic en: del_dependencias.bat
2. Lee la lista de lo que se va a eliminar
3. Presiona "S" para confirmar o "N" para cancelar
4. Espera a que termine la limpieza
5. ¡Tu computadora quedará como antes de instalar la aplicación!

NOTA: Si después quieres volver a usar la aplicación, 
solo ejecuta dependencias.bat otra vez.

PROBLEMAS COMUNES:
=================
- Si sale error de Python: instala Python desde python.org
- Si no carga la página: verifica que no tengas otro programa usando el puerto 8000
- Si sale error 404: asegúrate de ir exactamente a http://localhost:8000
- Si quieres empezar de cero: usa del_dependencias.bat y luego dependencias.bat

¡Disfruta organizando tus tareas! 😊
